This module provides a button on each sale order line to view the stock
moves history related to the product of that sale order line. It
enhances the visibility of stock movements directly from the sales order
interface.

**Table of contents**

.. contents::
   :local:

Usage
=====

-  Navigate to Sales.
-  Create or open a sale order.
-  On each sale order line, next to the 'Delivered' field, you will find
   a button to view the stock moves history for the product in that sale
   order.
